# frontify-bootstrap

Sync Bootstrap patterns to Frontify

## Configuration

Create a `.env` file at root and fill in with theses variables

* `FRONTIFY_API_KEY`
* `FRONTIFY_PROJECT_ID`

## Synchronize

* `git clone https://github.com/frassinier/frontify-bootstrap.git`
* `npm insall`
* `node index.js`
